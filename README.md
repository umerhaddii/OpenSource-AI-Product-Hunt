### Submission Guidelines

1. **Check Criteria**:
   - Must be open source (GitHub/GitLab with OSI-approved license)
   - Must have 100+ upvotes on Product Hunt
   - Must be tagged as "Developer Tools" on PH
   - Should be AI-related (AI-powered, ML tools, LLM integrations, etc.)

2. **Submit via Issues**:
   - Create a new issue with the title: `[Tool Submission] Tool Name`
   - Include:
     - GitHub/GitLab repository URL
     - Product Hunt URL
     - Brief description (one sentence)
     - Current upvote count
     - Month it was featured/launched

3. **Pull Requests**:
   - Add your tool to the appropriate month section
   - Follow the existing format
   - Keep entries sorted by upvotes (highest first)
   - Ensure all links work
